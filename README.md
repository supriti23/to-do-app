# To-Do App
 This is a frontend application where you can add and delete tasks from your To-Do list. You can also filter the tasks added by selecting the filter options as "To-Do" or "done" or "All". There is checkbox to mark each tasks as "done". Upon adding a new task it appears as a card below which has an option to mark as done and also delete.

## Technologies Used
- TypeScript
- Tailwind CSS
- Zustand(React State Management)
- React JS
- Jest and React Testing Library


## System Requirements
Yarn
Node.js
Here is the link to install node.js in your system - https://nodejs.org/en/download/
## Running the Application
Open terminal in the project directory, you can run:

### `yarn install`
All dependencies are installed to run the application.

### `yarn start`
Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `yarn test`
Launches the test runner in the interactive watch mode.\
See the section about [running tests](https://facebook.github.io/create-react-app/docs/running-tests) for more information.

### `yarn build`
Builds the app for production to the `build` folder.\
It correctly bundles React in production mode and optimizes the build for the best performance.
